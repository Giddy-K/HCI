# KRA iTax Redesign — Redline UX Collective

A Human-Computer Interaction (HCI) project by a student team at Jomo Kenyatta University of Agriculture and Technology (JKUAT). The project applies UX research and design principles to redesign the Kenya Revenue Authority's iTax system, making it more usable, accessible, and intuitive for Kenyan taxpayers.

## Live Pages

## Running Locally

No build step required — open any `.html` file directly in a browser, or serve the directory with any static file server:

```bash
npx serve .
# or
python -m http.server 8080
```

## Project Structure

```
├── index.html              # Portfolio home
├── itax-redesign.html      # Main case study
├── auth.html               # Prototype: login
├── dashboard.html          # Prototype: dashboard
├── file-returns.html       # Prototype: filing
├── payment.html            # Prototype: payments
├── otp.html                # Prototype: OTP
├── account-switcher.html   # Prototype: account switcher
├── styles.css              # Portfolio styles
├── system-styles.css       # Prototype system styles
├── script.js               # Portfolio scripts
├── system-script.js        # Prototype shared scripts (demo data)
└── assets/                 # Images and icons
```

> **Note:** `system-script.js` uses client-side `localStorage` for session management. This is intentional for demo purposes and not intended for production use.
